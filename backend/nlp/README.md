# BrevityNLP
### Windows
```
python -m venv c:\virtualenvs\PYTHON3
c:\virtualenvs\PYTHON3\Scripts\activate 
cd BrevityNLP
pip install -r requirements.txt -U
```

### Mac
```
python3 -m venv ~/virtualenvs/PYTHON3
source ~/virtualenvs/PYTHON3/bin/activate 
cd BrevityNLP
pip install -r requirements.txt -U
```
